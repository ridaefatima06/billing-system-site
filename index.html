Billing System

Software Construction & Development Lab

Project Description

This Billing System is a desktop application developed using Python (CustomTkinter) and Microsoft Access database. It allows users to manage customers, products, purchases, and generate billing receipts automatically.

Core Features

  • Add customer purchase records
  • Auto product price fetching from database
  • Generate detailed receipt with totals
  • Apply discount on high purchases
  • Search and filter products
  • Database integration using MS Access

Technology Used

  • Python (CustomTkinter)
  • PyODBC
  • Microsoft Access Database
  • SQL Queries
